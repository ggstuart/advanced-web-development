> We can now use the `align-content` property in block elements.

```html
<div>This is just a div</div>
```

```css
div {
    background: yellow; 
    text-align: center;
    align-content: center; 
}
```

<div style="align-content: center; background: yellow; text-align: center;">This is just a div</div>
